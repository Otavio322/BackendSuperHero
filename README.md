# Backend SuperHero 🦸‍♀️

Este é o **backend** da aplicação SuperHero.  
Ele fornece uma API REST para gerenciar heróis (CRUD).

## 🚀 Tecnologias
- Node.js
- Express
- MongoDB (Atlas)
- Mongoose
- CORS

## ⚙️ Rotas disponíveis
- `GET /api/entries` → Lista todos os heróis
- `GET /api/entries/:id` → Busca herói por ID
- `POST /api/entries` → Cria novo herói
- `PUT /api/entries/:id` → Atualiza herói
- `DELETE /api/entries/:id` → Remove herói
